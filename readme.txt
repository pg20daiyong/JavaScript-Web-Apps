## Synopsis
---------------
Programming for Games, Web and Mobile Course
- T<2> JavaScript Web Apps
- Assignment - 1
This application is the Level Editor of Angry Bird game.
You can complete the Level Editor by Drag, Drop and fill out the content.
And you can save the game related json files
If the completion of this game. using the Json file you can play this game.

To use this program, you have to install node package and below commands are necessary.

npm init
npm install express
npm install path
npm install http
npm install nodemon

To run this program, you have to read "how to use"
